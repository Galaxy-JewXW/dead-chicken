# 随机危险树木标记改进说明

## 修改概述

根据用户反馈，将随机标记的危险树木从简单的文字显示改为使用标准的危险物标记系统，提供更好的视觉效果和一致性。

## 主要修改

### 1. 修改 `MarkTreeAsDangerous` 方法

**之前的实现：**
- 使用简单的 `TextMesh` 显示危险等级文字
- 使用自定义的 `SimpleBillboard` 组件使文字面向摄像机
- 使用 `MarkerPrefab` 组件（功能有限）

**现在的实现：**
- 使用标准的 `DangerMarker` 组件
- 自动创建三角形警示标志和感叹号
- 使用内置的 `Billboard` 组件使标记面向摄像机
- 支持不同危险等级的视觉区分

### 2. 添加危险等级转换

新增 `ConvertTreeDangerLevel` 方法，将 `TreeDangerMonitor.TreeDangerLevel` 转换为 `DangerLevel`：

- `Safe` → `Low`
- `Warning` → `Medium` 
- `Critical` → `High`
- `Emergency` → `High`

### 3. 删除冗余组件

- 删除了 `SimpleBillboard` 类，因为 `DangerMarker` 已经包含 `Billboard` 功能
- 简化了代码结构

## 视觉效果改进

### 标记外观
- **三角形警示标志**：根据危险等级显示不同颜色
  - 低危险：金黄色
  - 中等危险：橙红色  
  - 高危险：深红色（带发光效果）
- **感叹号**：白色或黑色，根据背景色自动调整对比度
- **自动面向摄像机**：标记始终面向玩家视角

### 树木高亮
- 危险树木的材质会添加发光效果
- 发光颜色与危险等级对应
- 发光强度适中，不会过于刺眼

## 功能特性

1. **重复标记防护**：检查树木是否已有标记，避免重复创建
2. **类型标识**：所有随机标记的危险树木都标记为 `Vegetation` 类型
3. **描述信息**：包含"随机标记的危险树木"前缀，便于识别
4. **创建者标识**：标记为"随机检测系统"创建

## 使用方法

1. 在树木监测面板中点击"🎲 随机标记危险"按钮
2. 系统会自动选择5-15%的树木进行随机标记
3. 标记的树木会显示标准的危险物标记（三角形+感叹号）
4. 树木本身会有发光高亮效果

## 技术细节

- 使用 `DangerMarker.SetDangerInfo()` 方法设置标记信息
- 标记位置设置在树木上方2米处
- 支持所有现有的危险物标记功能（点击、跳转等）
- 与现有的危险物管理系统完全兼容
